# BAD
### Basic Archive Downloader

> **⚠️ Этот проект сделан исключительно по фану с помощью Claude. Не воспринимайте его всерьез**

BAD — минималистичный пакетный менеджер для **CC: Tweaked** (ComputerCraft), вдохновлённый `apt`, `pacman` и другими менеджерами пакетов.

Он позволяет устанавливать, обновлять и удалять программы из собственных репозиториев прямо внутри игры.

---

## ✨ Возможности

- 📦 Установка пакетов
- 🔄 Обновление программ
- 🗑️ Удаление пакетов
- 🔍 Поиск по репозиториям
- 📋 Просмотр информации о пакете
- 📚 Поддержка нескольких репозиториев
- 🔗 Автоматическая установка зависимостей
- 💾 Локальный кэш индекса

---

# 🚀 Установка

## Через интернет (рекомендуется)

```lua
wget https://raw.githubusercontent.com/hez1ch/bad/main/badpm/install.lua installer
installer
```

После установки команда станет доступна как

```lua
bad
```

---

## Ручная установка

Просто скопируйте

```
badpm/bad.lua
```

в

```
/bin/bad
```

и выдайте права на запуск (если требуется вашей системой).

---

# 🌐 HTTP

BAD скачивает файлы через HTTP API ComputerCraft.

На сервере необходимо включить HTTP в

```
config/computercraft-server.toml
```

```toml
[http]
enabled = true
```

Также не должен блокироваться

```
raw.githubusercontent.com
```

# 📦 Команды

| Команда | Описание |
|---------|----------|
| `bad update` | обновить список пакетов |
| `bad install <pkg>` | установить пакет |
| `bad remove <pkg>` | удалить пакет |
| `bad upgrade` | обновить все пакеты |
| `bad list` | список установленных |
| `bad search <text>` | поиск |
| `bad info <pkg>` | информация о пакете |
| `bad repo add <url>` | добавить репозиторий |
| `bad repo remove <url>` | удалить репозиторий |
| `bad repo list` | список репозиториев |

# 📦 Формат packages.json

```json
{
  "packages": {
    "hello": {
      "version": "1.0.0",
      "description": "Example package",
      "author": "Author",
      "depends": [],
      "files": [
        {
          "url": "https://...",
          "path": "/bin/hello"
        }
      ]
    }
  }
}
```

# ⚙️ Как работает

BAD хранит свои данные в

```
/.bad/
```

Там находятся:

- список репозиториев;
- локальный индекс пакетов;
- информация об установленных программах.

Во время установки менеджер:

1. скачивает индекс;
2. ищет пакет;
3. устанавливает зависимости;
4. скачивает файлы;
5. сохраняет информацию для дальнейшего обновления или удаления.

# ➕ Создание собственного репозитория

Создайте файл

```
packages.json
```

и разместите его где угодно (например, на GitHub через Raw).

После этого добавьте его:

```lua
bad repo add https://example.com/packages.json
```

или

```lua
bad repo add https://raw.githubusercontent.com/username/repo/main/packages.json
```
